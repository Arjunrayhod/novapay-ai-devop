# v0.5.0 — Kubernetes

**Release Date:** 2026-05-06

## Features

- Kubernetes integration via official Python client
- Pod, deployment, service, namespace, node, event, configmap, secret management
- 20+ API endpoints under `/api/kubernetes`
- Frontend K8s Center with resource tables, pod logs, event timeline
- Namespace-scoped and cluster-scoped resource queries

## Architecture

- `k8s/` module with lazy singleton K8s API client
- Support for in-cluster and kubeconfig-based authentication
- Graceful degradation when cluster is unreachable
- Frontend: `modules/kubernetes/` with resource filtering and search
