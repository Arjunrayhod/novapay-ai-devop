# v0.3.0 — Environment Module

**Release Date:** 2026-04-15

## Features

- Environment detection system (OS, CPU, memory, disk, network)
- Module router with 6 endpoints under `/api/environment`
- Frontend environment overview page with system info cards
- System, Python, Network, Storage, Process, Security detectors
- Router-level and module-level separation (`/api/v1/environment` + `/api/environment`)

## Architecture

- **Pattern established:** Services directory with plain sync functions returning `dict`
- **Pattern established:** Router with Pydantic schema validation
- Detector pattern: `BaseDetector` abstract class, concrete implementations per domain
- Frontend: `modules/environment/` with types, services, hooks, components, pages

## Endpoints

| Endpoint | Description |
|---|---|
| `GET /api/environment/overview` | System overview |
| `GET /api/environment/python` | Python runtime info |
| `GET /api/environment/packages` | Installed packages |
| `GET /api/environment/variables` | Environment variables |
| `GET /api/environment/diagnostics` | Full diagnostics |
| `GET /api/environment/health` | Service health |
