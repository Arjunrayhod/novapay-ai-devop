# v0.2.0 — Design System

**Release Date:** 2026-04-08

## Features

- `@aegisai/ui` shared component library with Tailwind CSS
- DashboardLayout with sidebar navigation
- PageContainer, PageHeader, SectionHeader layout components
- Card, GlassCard, MetricCard, StatusBadge data display components
- Search input component
- Typography system, color palette, spacing scale
- Dark mode support via Tailwind dark variant

## Architecture

- UI components built on Tailwind CSS utility classes
- GlassCard uses backdrop-blur for frosted-glass effect
- MetricCard pattern reused across all monitoring modules
- StatusBadge uses semantic colors (success, warning, danger, neutral)

## Validation

- All components render with both light and dark themes
- TypeScript strict mode passes
- Tree-shakeable exports from `@aegisai/ui`
