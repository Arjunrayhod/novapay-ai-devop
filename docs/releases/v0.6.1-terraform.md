# v0.6.1 — Terraform

**Release Date:** 2026-06-15

## Features

- Terraform CLI integration via subprocess
- State inspection, module listing, resource tree, provider list
- Plan analysis (add/change/destroy counts)
- Output viewer with sensitive value masking
- 8 API endpoints under `/api/terraform`
- Frontend Terraform Center with state viewer, module table, plan viewer

## Architecture

- `terraform_center/` module following established pattern
- `_run_terraform()` utility with Windows STARTUPINFO support
- Lazy singleton for CLI availability check
- Major bug fix: `app.include_router()` was missing for terraform_module_router, fixed during architecture audit
- Frontend: `modules/terraform/` with AI panel, health panel, quick actions
