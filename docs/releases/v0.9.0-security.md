# v0.9.0 — Security Center

**Release Date:** 2026-07-05

## Overview

Sprint 9 introduces the Security Center — a comprehensive security dashboard with SAST scanning, dependency auditing, vulnerability management, policy evaluation via OPA, and compliance reporting.

## Backend: 8 API Endpoints

| Endpoint | Description |
|---|---|
| `GET /api/security/health` | Security tool availability |
| `GET /api/security/overview` | Aggregate security posture |
| `GET /api/security/sast/scan` | Run SAST scan on codebase |
| `GET /api/security/dependencies` | Dependency audit results |
| `GET /api/security/vulnerabilities` | Vulnerability summary |
| `GET /api/security/policies` | List OPA policies |
| `GET /api/security/policies/evaluate` | Evaluate OPA policies |
| `GET /api/security/compliance` | Compliance report |

## Features

### SAST Scanning
- Built-in Python AST analyzer detects dangerous functions (eval, exec, pickle), hardcoded secrets, SQL injection patterns, and assert statements
- Optional bandit integration for deeper analysis when CLI is available
- Combined result set deduplicated by file/line/check_id

### Dependency Audit
- Parses `requirements.txt`, `requirements-dev.txt`, `pyproject.toml`, and `package.json`
- Built-in vulnerability database: 14+ popular Python packages with known CVEs
- Optional safety CLI integration for expanded coverage

### Vulnerability Management
- Trivy CLI integration for filesystem scanning
- Severity breakdown (critical/high/medium/low)

### Policy Engine
- OPA policy discovery from `/policies/` directory
- Rego file detection, OPA evaluation when CLI available
- Graceful degradation: reports OPA as unavailable without CLI

### Compliance Reporting
- 4 compliance frameworks: SAST, Dependency Audit, Policy as Code, Secret Scanning
- Overall score calculation (percentage)
- Per-framework status and score

## Frontend

- Security Center page with overview cards (8 MetricCards)
- AI Insights panel with rule-based security recommendations
- Tool health panel with StatusBadge for 5 tools
- SAST results panel with severity-coded issue list (high/medium/low)
- Dependency audit panel with vulnerability list and CVE references
- Compliance panel with progress bar and per-framework status
- Quick actions: Refresh All, Run SAST Scan
