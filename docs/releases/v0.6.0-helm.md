# v0.6.0 — Helm

**Release Date:** 2026-06-01

## Features

- Helm CLI integration via subprocess
- Release management: list, inspect, history, values
- Chart management: search, list repositories, show dependencies
- 10 API endpoints under `/api/helm`
- Frontend Helm Center with release table, chart browser, dependency graph
- Health check with graceful degradation when Helm CLI is unavailable

## Architecture

- `helm_center/` module following established pattern
- `_run_helm()` utility in `utils.py` with Windows STARTUPINFO support
- Lazy singleton pattern for CLI availability check
- Cached `_helm_available` state to avoid repeated CLI calls
- Frontend: `modules/helm/` with release details, history timeline

## Endpoints

| Endpoint | Description |
|---|---|
| `GET /api/helm/version` | Helm CLI version |
| `GET /api/helm/health` | Helm health summary |
| `GET /api/helm/overview` | Aggregate overview |
| `GET /api/helm/releases` | List releases |
| `GET /api/helm/releases/{name}` | Release detail |
| `GET /api/helm/history/{name}` | Release history |
| `GET /api/helm/charts` | Search charts |
| `GET /api/helm/repositories` | List repos |
| `GET /api/helm/values/{release}` | Release values |
| `GET /api/helm/dependencies/{chart_ref}` | Chart dependencies |
