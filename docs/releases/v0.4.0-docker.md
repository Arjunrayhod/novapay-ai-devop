# v0.4.0 — Docker Engine

**Release Date:** 2026-04-22

## Features

- Docker Engine integration via official Docker SDK for Python
- Container management: list, inspect, start, stop, restart, logs
- Image management: list, inspect, pull, remove, prune
- System info: version, disk usage, events, stats
- 15 API endpoints under `/api/docker`
- Frontend Docker Center page with container table, image gallery, system dashboard

## Architecture

- Docker SDK client as lazy singleton in `utils.py`
- `docker_engine/` module with 6 service files
- Frontend: `modules/docker/` with container logs viewer, stats charts
- Live container stats via polling (10s interval)

## Endpoints

| Endpoint | Description |
|---|---|
| `GET /api/docker/containers` | List all containers |
| `GET /api/docker/containers/{id}` | Inspect container |
| `POST /api/docker/containers/{id}/{action}` | Start/stop/restart |
| `GET /api/docker/containers/{id}/logs` | Container logs |
| `GET /api/docker/images` | List images |
| `GET /api/docker/images/{id}` | Inspect image |
| `POST /api/docker/images/pull` | Pull image |
| `DELETE /api/docker/images/{id}` | Remove image |
| `POST /api/docker/images/prune` | Prune unused images |
| `GET /api/docker/version` | Docker version |
| `GET /api/docker/info` | System info |
| `GET /api/docker/disk` | Disk usage |
| `GET /api/docker/events` | Live events |
| `GET /api/docker/stats` | Container stats |
| `GET /api/docker/health` | Docker health |
