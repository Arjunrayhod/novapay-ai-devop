# v0.1.0 — Foundation

**Release Date:** 2026-04-01

## Features

- Project scaffold with FastAPI backend and Next.js frontend
- Monorepo structure with pnpm workspaces
- Docker Compose development environment
- API health endpoint at `/api/health`
- Basic settings and configuration module
- CORS middleware for local development

## Architecture

- **Backend:** FastAPI (Python) in `dashboard/apps/api/`
- **Frontend:** Next.js (TypeScript) in `dashboard/apps/web/`
- **Packages:** Shared UI, utils, types, theme, hooks in `dashboard/packages/`
- **Infrastructure:** Docker Compose for local dev

## Validation

- Python project initialized with pyproject.toml
- Next.js project initialized with TypeScript
- pnpm workspace configured with 7 packages
